# Quarkz

An encryption library


## Installing

For development, it is recommended to use a virtual env: 
```
$ python3 -m venv env
```
To activate it: 
```
$ source env/bin/activate
```
or when using the fish shell:
```
$ source env/bin/activate.fish
```

To install:
```
$ pip3 -r requirements.txt
```

## Run The Sample 
```
$ python3 -m quarkz.aes
```
